Klaviyo-Webflow Integration Example

Email Subscription Form

This example shows how to use the Klaviyo-Webflow integration with a simple email subscription form.

Thanks for subscribing! Check your email for confirmation.
Oops! Something went wrong. Please try again.

How It Works

The form above is configured with the following attributes:

<form id="email-form" data-klaviyo-form data-klaviyo-list-id="YOUR_LIST_ID" data-klaviyo-account-id="YOUR_PUBLIC_API_KEY">

These attributes tell the integration:

  • data-klaviyo-form: This form should be processed by the Klaviyo integration
  • data-klaviyo-list-id: The Klaviyo list to subscribe users to
  • data-klaviyo-account-id: Your Klaviyo public API key (required)

Advanced Configuration

You can also configure the integration with JavaScript. Here's an example:

// Initialize with JavaScript configuration window.initKlaviyoWebflow({ publicApiKey: 'YOUR_PUBLIC_API_KEY', defaultConfig: { listId: 'YOUR_DEFAULT_LIST_ID', tracking: { viewForm: true, submitForm: true }, debug: true }, forms: { 'email-form': { listId: 'CUSTOM_LIST_ID_FOR_THIS_FORM', customProperties: { source: 'example page' } } } });
